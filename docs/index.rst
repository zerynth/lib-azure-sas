****************************************
Microsoft Azure Shared Access Signatures
****************************************

`Microsoft Azure Shared Access Signatures <https://docs.microsoft.com/en-us/azure/storage/common/storage-dotnet-shared-access-signature-part-1>`_ provide a way to grant limited access to Microsoft Azure objects in a storage account to other clients, without exposing account keys.
